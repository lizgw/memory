Memory

It's been a while since I made a game, so I decided to create something simple yet fun: a memory game! This was a lot of fun to program, and I got to use a lot of the concepts that I have been learning in my computer science class. I'm pretty proud of the end result! Thanks for playing!

How to Play

Click a tile to turn it over, then find the matching card. Try to find all of the matches in as few moves as possible!
